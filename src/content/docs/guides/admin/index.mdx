# Setup
Using the plugin`s [jar](https://github.com/TastyPommesLul/BCBlock/releases) file, you can install the plugin on your server.

Going on the link above, you can download the latest version (or any other), that you want.
Put the jar file in your Server's `plugins` folder.

From there, your all set and can move on to the next Topic, Configuration.
